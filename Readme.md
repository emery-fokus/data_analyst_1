#  Analyse des données e-commerce – Data Analyst 1

Ce projet explore un jeu de données e-commerce contenant des informations sur des produits Amazon. L’objectif est de nettoyer les données, extraire des insights clés et visualiser les tendances par marque, catégorie et année d’ajout.

##  Structure du projet

- `main.py` : script Python principal pour l’analyse
- `E-commerce_data.xlsx` : fichier de données source
- `README.md` : documentation du projet

##  Étapes de l’analyse

1. Chargement et nettoyage des données
2. Conversion des dates et suppression des doublons
3. Création d’une colonne `annee_ajout`
4. Visualisation :
   - Nombre de produits ajoutés par année
   - Marques les plus présentes
   - Catégories les plus fréquentes

##  Exemples de visualisations

-  Histogramme des ajouts par année
-  Top 10 des marques les plus présentes
-  Top 10 des catégories les plus fréquentes

##  Technologies utilisées

- Python
- Pandas
- Matplotlib
